AI Lab Work - Machine Learning Model

This project involved building a **machine learning model** to predict customer churn for a company. The goal was to understand customer behavior and reduce churn rates by identifying patterns. The model was trained using **scikit-learn**, and I faced challenges such as **data imbalance**. To address this, I implemented **SMOTE** (Synthetic Minority Over-sampling Technique) to balance the dataset and improve model accuracy.

Reflection on AI Lab Work - Machine Learning Model

Creating this machine learning model deepened my understanding of the challenges in real-world AI projects. I learned valuable lessons in **data preprocessing**, especially handling imbalanced datasets, and **algorithm optimization**. By working through these challenges, I gained hands-on experience that will be crucial for my future work in AI/ML. This artifact demonstrates my ability to apply AI/ML techniques to solve practical problems and shows my growth as a developer and data scientist.

Artifact 3 - Deep Learning Model for Image Classification

This project involved building a **Deep Learning model** using **TensorFlow** to classify images from a large dataset. The model utilized a **Convolutional Neural Network (CNN)** to extract features from images, allowing it to categorize them into predefined classes like **cats, dogs, and other animals**. One of the key challenges in this project was optimizing the model's **hyperparameters** to achieve high accuracy while avoiding **overfitting**.

Reflection on Deep Learning Model for Image Classification

Working on this **deep learning project** was an exciting challenge. It allowed me to explore **CNNs**, one of the most powerful techniques for **image recognition**. I learned how to fine-tune **hyperparameters** and overcome challenges such as **overfitting** by using techniques like **dropout** and **data augmentation**. This project significantly expanded my skills in **deep learning** and gave me a deeper understanding of **AI models** and their real-world applications in fields like **computer vision**.
