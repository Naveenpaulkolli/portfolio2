Artifact 4 - Machine Learning Model for Predicting Customer Churn

This artifact showcases a machine learning model I developed to predict **customer churn**. The model uses historical customer data to identify patterns and predict when a customer is likely to leave, allowing businesses to proactively engage at-risk customers and improve retention rates.

Objective

The goal of this project was to build a reliable model for predicting customer churn. By identifying potential churn, businesses can take proactive measures to retain customers, ultimately improving customer loyalty and reducing turnover.

Process

  • Data Collection & Cleaning: I gathered customer data, cleaned it by removing missing values, and encoded categorical variables.
  • Feature Engineering: Selected features like customer age, usage frequency, and service plan to be used in the model.
  • Model Selection: I experimented with **Logistic Regression** and **Random Forest** classifiers.
  • Addressing Class Imbalance: Used **SMOTE (Synthetic Minority Over-sampling Technique)** to balance the dataset.
  • Model Evaluation: Evaluated the model using **cross-validation**, focusing on metrics like **precision** and **recall**.

Tools & Technologies Used

  • Python: For implementing the code.
  • scikit-learn: For model building and evaluation.
  • SMOTE: To address the class imbalance problem.
  • Jupyter Notebook: For documenting and visualizing the project.

Reflection

This project helped me deepen my understanding of **machine learning workflows**, especially in handling **imbalanced data** and improving **model performance**. I learned to carefully preprocess data, choose the right model, and evaluate using the correct metrics, which is crucial for building robust AI solutions.
